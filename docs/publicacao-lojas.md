# Publicacao nas lojas

## Android

1. Criar conta Google Play Console.
2. Rodar build do app.
3. Gerar projeto Android com Capacitor.
4. Gerar arquivo AAB no Android Studio.
5. Enviar para teste interno.
6. Preencher ficha da loja, seguranca de dados e politica de privacidade.
7. Enviar para revisao.

## iOS

1. Entrar no Apple Developer Program.
2. Abrir projeto iOS no Xcode.
3. Enviar build para App Store Connect.
4. Testar com TestFlight.
5. Preencher App Privacy, descricao, screenshots e suporte.
6. Enviar para revisao.

## Material necessario

- Politica de privacidade.
- Termos de uso.
- Icone 1024x1024.
- Screenshots.
- Descricao curta e longa.
- E-mail de suporte.
- Site ou pagina publica.
